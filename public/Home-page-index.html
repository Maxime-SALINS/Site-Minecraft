Le site de BacchuZz

Présentation du site

Bonjour, je suis BacchuZz, youtuber Mincraft, je vous souhaite la bienvenue sur mon site. Hâte de pouvoir partager mon univers avec vous.

Ce site a pour but de réunir la communauté de BacchuZz autour de Minecraft. Pour ceux qui ne connaissent pas ce jeu, Mincraft est un jeu de survie où l'on peut faire absolument tous ce que l'ont veut. Ce jeu est un partage entre l'exploration d'un monde quasiment infini, composé de différentes structures comme les villages, les temples, les cités anciennes etc... De survie puisqu'il faut faire attention a une multitude de monstres, ce qui rajoute du piquant à l'aventure ! C'est un aussi un jeu ou nous pouvons laisser exprimer notre créativité, puisqu'il y a la possibilité de construir ce que l'on veut à l'infini.

Sur ce site je vous partagerais les constructions que j'ai réalisées sur mes différentes parties, cela vous permettra de suivre mes aventure, mais aussi, de vous inspirer de mes constructions afin de les intégrer dans vos parties. Il y aura aussi quelque tuto sur la redstone, tuto qui vous aideront a faire de des usines afin de faciliter votre survie dans le jeu. Vous pourrez, aussi, partager vos expériences avec les autres joueurs de la communauté.

Onglet construction : Venez voir mes constructions que vous pourrez utiliser dans vos parties solo ou en multi !
